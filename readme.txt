== Description ==

What is Visibility Viewer?

Visibility viewer is a simple plugin that handle the private page visibility.
The default behaviour of a private page, is display the 404 page error if access by anonymous user.
Visibility viewer overwrite the page content with a custom error message if access by anonymous user.

== Installation ==

To install in regular WordPress:

1. Upload the `visibility-viewr` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. Now logout from your wordpress site and open your private page, you should see the error message instructions
